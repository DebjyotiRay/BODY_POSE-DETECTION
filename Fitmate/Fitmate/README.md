### Run Locally

To run **Fitmate** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Wemofy/Fitmate.git
```

Windows:

```bash
git clone https://github.com/Wemofy/Fitmate.git
```
